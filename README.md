# Prostate-Cancer-survival-analysis
## Code and results for PCa manuscript clinical significance section. 

Description: The analysis is done on 2 major axes. 1) genetic/intrinsic heterogeneity drivers as found from feature ranking on cell fate prediction model. 2) extrinsic heterogeneity drivers studied in ABM - a) androgen uptake and synthesis, b) EMT-MET/adhesion-motility-cytoskeletal regulators.
